＜画面表示を見やすくする設定＞
SET LINESIZE 160
COLUMN ID FORMAT "000" TRUNCATE
COLUMN NAME FORMAT a20 TRUNCATE
COLUMN ZIP FORMAT a10 TRUNCATE
COLUMN ADDRESS1 FORMAT a30 TRUNCATE
COLUMN ADDRESS2 FORMAT a15 TRUNCATE
COLUMN TEL FORMAT a20 TRUNCATE
COLUMN FAX FORMAT a20 TRUNCATE
COLUMN EMAIL FORMAT a20 TRUNCATE

＜LINESIZEの確認方法（デフォルトは80）＞
SHOW LINESIZE

＜COLUMN … FORMAT …の取り消し方法＞
COLUMN {カラム名} CLEAR
